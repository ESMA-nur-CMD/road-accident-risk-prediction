# Accident Risk Prediction
In this project, I worked on accident-related data to understand which factors may affect accident risk and to build a simple machine learning model for prediction.

## Project Content
 Examining the dataset
 Cleaning missing or unnecessary data
 Preparing features for the model
 Training and testing the model
 Evaluating the results

## Technologies Used
  Python
  Pandas
  NumPy
  Matplotlib
  Scikit-learn

## Purpose
The main goal of this project is to practice machine learning steps on a real dataset and gain experience in data preprocessing and model evaluation.

## Dataset
The dataset was taken from Kaggle for educational purposes.
